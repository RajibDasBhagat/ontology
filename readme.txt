A list of computer scientists fetched from Wikidata is displayed on the html page. On seecting a particular person, the details of the person are listed on the same page on the right hand side. The links to the ACM profile, DBLP and VIAF profile of the person is also provided with which more details of the same person can be fetched.

Apache Jena is used for connecting to SPARQL end point and obtain the query results through Java.
